# 居中

# 水平居中

  1. 块级元素包裹行内元素(span,inline-block)，直接给外层 text-align: center
<style>
.p{
  border: 1px solid red;
}
.c{
  border: 1px solid black;
}
.text-center{
  text-align: center;
}
</style>
<div class="p text-center">
    <span class="c">水平居中</span>
</div>
  1. 块级元素包含有一定宽度的块级元素
<style>
.p{
  border: 1px solid red;
}
.c{
  border: 1px solid black;
}
.block{
  width: 300px;
  margin: 0 auto;
  text-align: center;
}
</style>
<div class="p">
    <div class="c block">水平居中</div>
</div>

3.flex

<style>
.p{
  border: 1px solid red;
}
.c{
  border: 1px solid black;
}
.flex-center{
  display: flex;
  justify-content: center;
}
</style>
<div class="p flex-center">
    <div class="c">水平居中</div>
</div>
  1. transform基于父级居中,但是父级会塌陷
<style>
.p{
  border: 1px solid red;
}
.c{
  border: 1px solid black;
}
.relative{
  position:relative;
}
.transform{
  position: absolute;
  left: 50%;
  transform: translateX(-50%);
}
</style>
<div class="p relative">
    <div class="c transform">水平居中</div>
</div>

5.绝对定位元素也可以使用margin，兼容更加好

<style>
.p{
  border: 1px solid red;
}
.c{
  border: 1px solid black;
}
.relative{
  position:relative;
}
.absolute{
  position: absolute;
  left: 0;
  right: 0;
  top: 0;
  bottom: 0;
  margin: auto;
  width: 4em;
}
</style>
<div class="p relative">
    <div class="c absolute">水平居中</div>
</div>

# 垂直居中

1.单行文本采用line-height

<style>
.p{
  border: 1px solid red;
}
.vertical{
  height: 50px;
  line-height: 50px;
}
</style>
<div class="p vertical">
    垂直居中
</div>

2.flex

<style>
.p{
  border: 1px solid red;
}
.flex-vertical{
  height: 50px;
  display: flex;
  align-items: center;
}
</style>
<div class="p flex-vertical">
    垂直居中
</div>

3.transform基于父级居中

<style>
.p{
  border: 1px solid red;
  width: 100%;
  height: 100%;
}
.c{
  border: 1px solid black;
}
.relative{
  position:relative;
}
.transform-v{
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}
</style>
<div class="p relative">
    <div class="c transform-v">垂直居中</div>
</div>

4.table-cell和vertical-align,可以多行文字居中

<style>
.p{
  border: 1px solid red;
  width: 100%;
  height: 100%;
}
.c{
  border: 1px solid black;
}
.table-cell{
  display: table-cell;
  height: 100px;
}
.vertical-align{
  vertical-align: middle;
}
</style>
<div class="p table-cell vertical-align">
    <div class="c">垂直居中</div>
    <div class="c">垂直居中</div>
</div>

# 水平垂直居中

大多数可以采用上面两者结合，介绍一个不常见的方法: flex结合margin（行内子元素慎用，国产手机系统可能会有坑）

<style>
.p{
  border: 1px solid red;
  width: 100%;
  height: 100px;
  display: flex;
}
.c{
  border: 1px solid black;
  margin: auto
}
</style>
<div class="p table-cell vertical-align">
    <div class="c">水平垂直居中</div>
</div>
